## 🛠️ Troubleshooting

### The app does not open
- Check that the files finished downloading
- Extract the zip file first
- Run the app as Administrator
- Make sure Windows Defender did not block the file

### My documents do not show up
- Confirm that the folder contains supported file types
- Check that the folder path is correct
- Refresh or rebuild the index
- Move the files to a simple folder path like `C:\personalforge\docs`

### Answers are weak or off-topic
- Add more text from the right topic
- Remove scanned PDFs with no readable text
- Use smaller, focused document sets
- Reindex after changing the files

### The app feels slow
- Close other heavy apps
- Use fewer large documents at first
- Give the tool more RAM if your PC has it
- Use a GPU if the training option supports it

## 📌 Suggested folder layout

A simple folder structure can help:
- `C:\personalforge\docs`
- `C:\personalforge\models`
- `C:\personalforge\output`
- `C:\personalforge\exports`

This keeps your files easy to find and helps avoid confusion later.

## ⌨️ File names that help

Use clear names like:
- `policy_handbook.pdf`
- `meeting_notes.txt`
- `project_overview.docx`
- `sales_data.csv`

Avoid names like:
- `final2newnew.pdf`
- `doc1.docx`
- `scan12345.pdf`
